# Web3 常见应用：Uniswap、Polymarket、Aave 还有哪些？

上一篇讲了现货、合约和交易所怎么挣钱，这篇就讲**链上真正在用的应用**——DEX、预测市场、借贷、NFT、流动性质押、域名、去中心化合约和社交等。你之前听过的 Uniswap、Polymarket、Aave、Curve、Lido、OpenSea、ENS、dYdX……到底是干啥的、和前面讲的 LP、Trader、代币咋对上号，捋一遍就清楚。

**先说明一点**：这些应用都在链上（主要是以太坊及 L2），**无许可**——不用注册、不用 KYC，连上钱包就能用。谁在里边赚钱、谁在里边亏钱，和之前说的角色（LP、Trader、Builder）都能对上。

---

## Uniswap — 去中心化兑换（DEX）

**干啥的**：链上的「换币」应用，用池子（流动性池）撮合买卖，没有订单簿、没有中心化托管。

你连上钱包，选两种币（比如 ETH / USDT），按当前池子里的价格**兑换**——你付出一边、收到另一边，币直接进你钱包。**没有交易所「帮你拿着」**，也没有挂单吃单，价格由 **AMM（自动做市商）** 公式算出来：池子里两种币的比例变了，价格就变。往池子里放币的人就是 **LP**，赚手续费分成；换币的人就是 **Trader**，付一点手续费。Uniswap 是最有名的 DEX 之一，很多新币首发也在这上边，**无许可上币**——谁都能建池子，所以机会和风险都有。

**一句话**：Uniswap = 链上换币的 DEX，用流动性池 + AMM 定价，LP 赚手续费，你拿钱包直接兑。

---

## Polymarket — 预测市场（Prediction Market）

**干啥的**：链上的「押事件结果」应用，对赌某件事会不会发生（例如大选、加息、某币能否破某个价），用市场价表示概率，结算在链上。

你买「会发生」或「不会发生」的份额，价格在 0～1 之间，**越接近 1 表示市场越认为会发生**。到期后按真实结果结算：押对了拿回 1，押错了归零。和传统博彩不同的是：**市场价本身就是「集体预测」**，很多人拿来当信息参考；和 CEX 合约不同的是：赌的是「事件结果」而不是「价格涨跌」。Polymarket 用 USDC 结算，部分在 Polygon 上，Gas 便宜。谁在里边赚钱？**信息更准、押对的人**；做市、提供流动性的人也能赚价差和手续费。

**一句话**：Polymarket = 链上押事件结果，市场价即概率，结算在链上，信息可当参考。

---

## Aave / Compound — 借贷协议

**干啥的**：链上的「存币生息、借币付息」应用，你把币存进去当**存款**拿利息，或者**抵押**手里的资产借出别的币（如抵押 ETH 借 USDT），利率由供需（池子里借多借少）动态算出来。

**Aave** 和 **Compound** 是两条最常用的借贷协议，逻辑类似：你存进去的币被别人借走，借的人付利息，**利息大部分分给存款人**，协议抽一点。你要借币，得先抵押超额资产（如 150% 抵押率），否则会被**清算**——价格跌太多，系统帮你卖抵押品还债。所以存款人赚的是「借币人付的利息」和协议代币激励；借币人要么是加杠杆（借 USDT 买更多币），要么是短期周转。利率随市场变，有时高有时低。

**一句话**：Aave / Compound = 链上存币生息、抵押借币，利率由供需定，借多了会被清算。

---

## Curve — 稳定币 DEX

**干啥的**：专注**稳定币之间兑换**的 DEX，用专门设计的 AMM 曲线减少滑点，适合大额换 USDT/USDC/DAI 等。

Uniswap 适合波动大的币对，稳定币之间价差小、量大，用 **Curve** 更省手续费、滑点更小。LP 往稳定币池子里放币，赚手续费和协议代币激励。很多协议（包括 Aave、Lido）的稳定币池子也接在 Curve 上。**一句话**：Curve = 链上换稳定币的 DEX，滑点小、适合大额，LP 赚手续费。

---

## Lido / Rocket Pool — 流动性质押（Liquid Staking）

**干啥的**：你存 **ETH**，拿 **stETH**（或 rETH）凭证，既能赚质押收益，又能把 stETH 拿去别的协议里用（借贷、做 LP），不用锁死等主网解锁。

以太坊改成 PoS 后，自己跑节点门槛高，**Lido**、**Rocket Pool** 这类协议帮你「代质押」：你存 ETH，他们汇总给节点，给你 1:1 的 stETH/rETH，**质押收益按比例发给你**。stETH 可以在 Aave、Curve 里当抵押品或做 LP，所以叫「流动性质押」。谁在里边赚钱？存 ETH 的人拿质押收益；协议抽一点；节点运营方拿分成。**一句话**：Lido / Rocket Pool = 存 ETH 拿 stETH 赚质押收益，还能把 stETH 拿去别处用。

## OpenSea / Blur — NFT 市场

**干啥的**：链上**买卖 NFT**（头像、艺术品、游戏道具等）的市场，你挂单、别人买，或者你直接买别人挂的，手续费和版税在链上结算。

**OpenSea** 是老牌 NFT 市场，支持多链；**Blur** 偏交易员、挂单吃单体验更接近交易所，一度用代币激励把交易量拉得很高。谁在里边赚钱？卖家赚价差；平台收手续费；创作者可以设**版税**（每次转手抽成）。NFT 价格波动大、流动性比代币差，玩的时候注意真假和合约安全。**一句话**：OpenSea / Blur = 链上买卖 NFT 的市场，平台收手续费，创作者可收版税。

---

## ENS（Ethereum Name Service）— 以太坊域名

**干啥的**：把一长串地址 `0x1234…abcd` 绑成一个**短名字**，比如 `alice.eth`，别人给你转钱、在应用里认你，都用这个名字，按年付费续期。

ENS 是链上的「域名 + 身份」：你注册一个 `.eth` 名字，链上记录「这个名字 → 你的地址」，钱包、DApp 都能解析。很多项目、KOL 用 ENS 当品牌和收款地址。**一句话**：ENS = 链上短域名，把地址变成 alice.eth，按年付费。

## dYdX / GMX — 去中心化永续合约

**干啥的**：在**链上**开永续合约（做多、做空、杠杆），不用把币存到中心化交易所，仓位和清算在链上或 L2 上执行。

**dYdX**（现多为 L2 订单簿）、**GMX**（用池子做对手方、无订单簿）是常见的去中心化合约应用。逻辑和 CEX 合约类似：保证金、杠杆、强平、资金费率；区别是**非托管**、透明、有的还能用协议代币激励做 LP 当对手方。**一句话**：dYdX / GMX = 链上开永续合约，非托管，做多/做空/杠杆和 CEX 类似。

---

## Lens / Farcaster — 去中心化社交

**干啥的**：链上的**社交图谱**：关注、发帖、点赞、评论等关系和数据在你自己的钱包或链上，不依赖单一公司服务器，应用可以读取同一套数据做前端。

**Lens Protocol**（多链）、**Farcaster**（偏以太坊生态）是两条常见的「去中心化社交」协议。你用钱包登录，关注谁、发了啥存在链上或去中心化存储，换一个前端（如 Warpcast、Lenster）也能看到同一套内容。**一句话**：Lens / Farcaster = 链上社交图谱，关注和发帖数据归你，换应用也能用。

## 顺带提几个常一起出现的

- **Gitcoin**：公共物品 / 开源项目资助，用二次方投票（Quadratic Funding）给项目捐钱，很多 Builder 和项目在这上边筹款。
- **Safe（Gnosis Safe）**：多签钱包，一笔转账要多人同意才执行，适合团队、DAO 管钱。
- **Chainlink**：预言机，把链下数据（价格、天气等）喂给链上合约，DeFi 的清算、价格大多靠它。
- **Snapshot**：链下投票、治理，很多 DAO 用 Snapshot 提案 + 投票，再按结果在链上执行。
- **L2 与跨链**：Uniswap、Aave 等都有在 Arbitrum、Optimism、Base、Polygon 上的版本，Gas 更便宜；跨链用桥（Bridge）或 CEX 中转，注意安全。
- **钱包**：用这些应用前要先有钱包（如 MetaMask、Rabby），自己保管助记词，链上操作都是自己签名、自己负责。
